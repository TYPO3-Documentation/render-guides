Examples of nesting
===================

.. highlight:: shell

.. rst-class:: bignums-xxl

1. Prepare

   .. rst-class:: bignums-important

   #. Check the requirements

      #. Machine accessible?
      #. Is `abc` installed? Run::

            which abc

      #. Is `bcd` available?

   #. Get yourself a coffee

   #. Stop everything else!


2. Install

   Now the actual stuff.

   .. rst-class:: bignums

   #. Abc

      #. Download from ...
      #. unpack
      #. run installer

   #. Bcd

      #. Download from ...
      #. unpack
      #. run installer

   #. Cde

      #. Download from ...
      #. unpack
      #. run installer


3. Cleanup

   **BEWARE:**

   .. rst-class:: bignums-warning

   #. Do not xxx!
   #. Do not yyy!
   #. Do not zzz!


4. Be a happy user!

   .. rst-class:: bignums-tip

   #. Run the stuff all day
   #. Run the stuff all night
   #. Never ever stop again
