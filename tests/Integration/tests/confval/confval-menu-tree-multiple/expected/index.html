array of cObjects ¶
Type
cObject

Array of cObjects. Use this to define cObjects for the different values of . If has a certain value, the according cObject will be rendered. The cObjects can have any name, but not the names of the other properties of the cObject CASE.

default ¶
Type
cObject

Use this to define the rendering for those values of cobj-case-key that do not match any of the values of the cobj-case-array-of-cObjects. If no default cObject is defined, an empty string will be returned for the default case.
